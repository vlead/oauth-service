1.2 Explanation

Flow of Information

4 Components :

1.2.1 Resource Owner

  • This is the User. The user wants to interact with the client(application)

1.2.2 Client

  • Client is the Application that wants access to the user's account. Before it receives access, the user must authorize it to do so.

1.2.3 Resource / Authorization Server

  • Protects the User's data and verifies the authenticity of the user. It then issues access tokens to the client.
